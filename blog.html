Linux命令大全（1）

arch 显示机器的处理器架构(1)

uname -m 显示机器的处理器架构(2)

uname -r 显示正在使用的内核版本

dmidecode -q 显示硬件系统部件 - (SMBIOS / DMI)(在mac上面测试，不管用)

hdparm -i /dev/hda 罗列一个磁盘的架构特性(在mac上面测试，不管用)

hdparm -tT /dev/sda 在磁盘上执行测试性读取操作(在mac上面测试，不管用)

cat /proc/cpuinfo 显示CPU info的信息(在mac上面测试，不管用)

cat /proc/interrupts 显示中断(在mac上面测试，不管用)

cat /proc/meminfo 校验内存使用(在mac上面测试，不管用)

cat /proc/swaps 显示哪些swap被使用(在mac上面测试，不管用)

cat /proc/version 显示内核的版本(在mac上面测试，不管用)

cat /proc/net/dev 显示网络适配器及统计 (在mac上面测试，不管用)

cat /proc/mounts 显示已加载的文件系统(在mac上面测试，不管用)

lspci -tv 罗列 PCI 设备 (在mac上面测试，不管用)

lsusb -tv 显示 USB 设备 (在mac上面测试，不管用)

date 显示系统日期

cal 2007 显示2007年的日历表(在mac上面测试，有问题)

date 041217002007.00 设置日期和时间 - 月日时分年.秒 （在mac上面测试，有问题）
